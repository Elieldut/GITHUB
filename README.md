# GITHUB
O GitHub é uma plataforma baseada na web que permite a desenvolvedores e equipes armazenar, gerenciar e colaborar em projetos de software usando o sistema de controle de versão Git. Funciona como uma rede social para programadores, onde é possível hospedar códigos, acompanhar alterações e interagir com outros profissionais globalmente. 
O que é e para que serve
O GitHub simplifica o processo de desenvolvimento colaborativo, permitindo que várias pessoas trabalhem no mesmo projeto simultaneamente sem sobrescrever o trabalho umas das outras. 
As principais funções incluem:
Hospedagem de Código: Armazena o código-fonte do projeto em "repositórios" na nuvem, tornando-o acessível de qualquer lugar.
Controle de Versão: Utiliza o Git para rastrear todas as alterações feitas nos arquivos ao longo do tempo. Isso permite reverter para versões anteriores, se necessário, facilitando a experimentação segura.
Colaboração: Facilita o trabalho em equipe através de funcionalidades como:
Branches (Ramificações): Cria cópias isoladas do código principal para desenvolver novos recursos ou corrigir bugs sem afetar o projeto original.
Pull Requests (Solicitações de Pull): Um mecanismo para propor alterações e solicitar que outros colaboradores as revisem, discutam e, eventualmente, mesclem (integrem) no projeto principal.
Issues (Problemas): Um sistema para relatar bugs, sugerir ideias e acompanhar o progresso das tarefas.
Networking e Aprendizado: É um espaço para aprender com projetos de código aberto (open source) de grandes empresas como Google e Microsoft, e também para construir um portfólio visível para a comunidade e potenciais empregadores.
